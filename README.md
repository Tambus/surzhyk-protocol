# Суржик

**Соціальний протокол автентичної української мови на блокчейні.**

> *Це не російська, це український суржик.*

---

## Що це

«Суржик» повертає українському суржику статус повноцінного саморізновида мови. Кожне суржикове слово карбується першим автором як NFT з обов'язковим зазначенням значення, регіону та прикладу. Кожне доповнення значень чи варіацій породжує геометрично-спадний потік роялті оригінальному карбувачу та попереднім контрибуторам, перетворюючи мовну спадщину на економічний актив.

## Проблема

Два століття суржик нав'язувався як «помилка» та «безкультурне мовлення», що породило мовну дискримінацію всередині українського суспільства та сприяло розділенню за регіонами, поколіннями і освітнім бекграундом. Жива регіональна мова, яку творять понад половина населення України, не має ні системного збереження, ні інструменту захисту від російської асиміляції, ні економічного стимулу для її носіїв.

## Як працює

### Шар 1 — Словник кластерів значень

- **Карб слова-вузла:** творець кластера сплачує `$SURZH`, on-chain записується центральне значення + літ. форма (нормативна укр., якщо є; інакше — першокарбована регіональна форма стає центром, як «ЛЕП»)
- **Перегляд безкоштовний:** browse словника відкритий, лічильник переглядів зважений по літопису переглядачів = чесний рейтинг популярності, стійкий до накрутки
- **Додавання регіональної форми:** контрибутор стейкає `$SURZH` на пропозицію додати свою форму у кластер; після схвалення вмикається у геометрично-спадний стек роялті (50/20/10/5/2.5%...)
- **Race condition:** перший on-chain commit при паралельних спробах; AI-кластеризація виявляє і пропонує merge дублюючих кластерів post-factum
- **Мапа регіонів:** hover на значення підсвічує всі регіони з відповідними формами

### Шар 2 — Літопис автора

- Soulbound non-transferable профіль
- Унікальні карби, схвалені доповнення, валідаційна активність → + до літопису
- Доведені фейки / спірні помилки → – літопис + refund комісій попередникам

### Шар 3 — Голос громадянина (vision)

- Авторизація через «Дію» з регіональним підтвердженням
- Вага голосу пропорційна літопису
- Інтеграція у міські DAO-голосування, культурні ініціативи, перевірку фактів

## Валідація

**Soft path:** голосування спільноти (+/–) для простих випадків.
**Hard path:** Підтверджувачі — філологи, краєзнавці, журналісти з підтвердженою експертизою, голос яких важить значно більше.
**Dispute forum:** спірні випадки виносяться у тред з аргументацією on-chain, з ескалацією до голосування або Підтверджувача.

## Технологічний стек

- **Smart contracts:** Solidity на BNB Smart Chain (з можливою міграцією операцій на opBNB)
- **Контракти:** ERC-721 (NFT слова), ERC-20 (`$SURZH`), кастомний розподільник роялті, soulbound літопис (ERC-5114)
- **Метадані:** IPFS (Pinata)
- **Індексація:** The Graph (on-chain), PostgreSQL (off-chain query, форум)
- **Фронт:** Next.js + TailwindCSS, wagmi/ethers.js, Web3Auth
- **AI-шар:** фонетична кластеризація варіацій (Sentence-Transformers / multilingual BERT)
- **Roadmap-інтеграції:** «Дія.Підпис», BlockPass
